## 🐛 常見問題

### 前端無法訪問
```bash
cd linkya-frontend
npm run dev
```

### 交易失敗
確認 Hardhat 節點運行中：
```bash
curl http://127.0.0.1:8545
```

### Dev 登入失敗
檢查控制台錯誤，確保 user-manager.ts 正確實現
